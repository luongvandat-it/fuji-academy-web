# Thư mục styles

Đặt biến CSS, theme, hoặc file CSS dùng chung (nếu cần).

- **`theme.css`** – Cấu hình màu dùng chung (primary #2BEEEE và các sắc độ). Được import trong `globals.css`. Dùng Tailwind: `bg-primary`, `text-primary-600`, `hover:bg-primary-600`, …
- **`globals.css`** – Nằm ở `src/app/globals.css` (Tailwind + global styles).
