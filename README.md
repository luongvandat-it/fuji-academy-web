# Fuji Web – Next.js App

Đây là thư mục chứa **toàn bộ mã nguồn** ứng dụng Next.js.

## Chạy nhanh

```bash
npm install   # lần đầu
npm run dev   # chạy dev → http://localhost:3000
```

## Cấu trúc code

- **`src/app/`** – Routes, layout, trang (App Router)
- **`src/components/`** – Components React (ui, layout, ...)
- **`src/lib/`** – Utils, constants
- **`src/hooks/`** – Custom hooks
- **`src/types/`** – Shared TypeScript types
- **`src/styles/`** – CSS dùng chung (nếu cần)

Chi tiết đầy đủ xem **README.md** ở thư mục gốc `FujiWeb/`.
